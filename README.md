# Capilarmente

Site sobre tratamentos para queda de cabelo no Brasil.

## Estrutura

- `/app` - Next.js App Router
- `/components` - Componentes React
- `/content` - Artigos em MDX
- `/lib` - Funções utilitárias
- `/types` - Tipos TypeScript

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Static export para `dist/`.
